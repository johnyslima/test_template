# Description

Верстка тестового макета.

___

### Installing

1. `git clone https://github.com/johnyslima/test_template.git`
2. `npm install`
3. `gulp`

___

### Dependencies

* `browser-sync`
* `del`
* `gulp`
* `gulp-cache`
* `gulp-cssnano`
* `gulp-if`
* `gulp-imagemin`
* `gulp-sass`
* `gulp-uglify`
* `gulp-useref`
* `gulp-autoprefixer`
* `run-sequence`
